# Alumni Portal

A platform to solve the fragmented alumni network problem starting with KEC (Kunwar's Engineering College), built as a scalable product for colleges and institutions.

## Project Vision

We are working on three layers simultaneously:

1. **KEC's Problem** — Build a functional alumni network for KEC, starting with data collection and connecting students with alumni.
2. **Product Thinking** — Design this generically from day one so it can scale to other colleges and institutions beyond KEC.
3. **AI-First Process** — Build using agentic, AI-native approaches. Not just using AI to code, but thinking about how AI fits inside the product too.
